# QR InfoBridge – Community Knowledge Tool Portugal

This is an early‑stage, community‑benefit project to design a **Portugal‑based QR code generator** that:
- Bridges digital and physical community spaces (e.g., streets, community centers, public events).  
- Is open source and does not collect personal data.  
- Is guided by values of transparency, decentralization, and local community benefit.
